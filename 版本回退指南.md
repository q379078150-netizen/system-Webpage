## 🔙 回退到v1.0.0的方法

### 方法一：使用Git标签回退（推荐）

#### 1. 查看v1.0.0标签

```powershell
# 查看所有标签
git tag

# 查看v1.0.0标签详情
git show v1.0.0
```

#### 2. 创建回退分支（安全方式）

```powershell
# 创建新分支指向v1.0.0
git checkout -b rollback-v1.0.0 v1.0.0

# 推送到远程
git push origin rollback-v1.0.0
```

#### 3. 切换到v1.0.0（临时查看）

```powershell
# 切换到v1.0.0标签
git checkout v1.0.0

# 查看代码
# 注意：这是detached HEAD状态，不能直接提交

# 返回主分支
git checkout main
```

#### 4. 重置到v1.0.0（危险，会丢失后续提交）

```powershell
# ⚠️ 警告：这会丢失v1.0.0之后的所有提交
git reset --hard v1.0.0

# 强制推送到远程（谨慎使用）
git push origin main --force
```

### 方法二：从GitHub Release下载

1. 访问：https://github.com/q379078150-netizen/system-Webpage/releases

### 方法三：使用GitHub Desktop

1. 打开GitHub Desktop
2. 点击 "History" 标签
3. 找到 v1.0.0 标签
4. 右键点击 → "Create branch from tag"
5. 分支名：`rollback-v1.0.0`
6. 切换到该分支
